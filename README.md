# Easy Webpack

An easy bundler referring to Webpack.

## Start

```bash
# clone with Git Bash
git clone https://github.com/jackluson/easy-babel.git

# change directory
cd easy-babel

# install dependencies
yarn install

# build for bundling
yarn run build
```
